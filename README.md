# Send Free Ticktes

## How to run
```
    git clone https://github.com/bitgoalsdeveloper/send_free_tickets.git
    cp .env.example .env
    ## update the key
    npm i
    npx hardhat run --network mainnet scripts/free-tickets.js

```
## How to run validate
```
    npx hardhat run --network mainnet scripts/fvalidate-tickets
```
